# 주의점
1. FTDI FT232RL 모듈로는 전력이 부족하여 WiFi 기능이 동작 안할 수 있음.


ESP8266-ESP01은 사물인터넷(IoT)과 임베디드 시스템 프로젝트에서 널리 사용되는 소형 Wi-Fi 모듈입니다. 주요 특징과 응용 분야는 다음과 같습니다:

# 하드웨어 사양
마이크로컨트롤러: ESP8266 SoC

플래시 메모리: 1MB 또는 2MB (버전에 따라 다름)

GPIO 핀: 2개 사용 가능 (GPIO0 및 GPIO2)

직렬 통신: UART (TX 및 RX 핀)

전원 공급: 3.3V

크기: 매우 소형

# 주요 특징
Wi-Fi 기능: 802.11 b/g/n 프로토콜 지원

저전력 소비: 배터리 작동 장치에 이상적

AT 명령어 세트: 쉬운 통신을 위한 사전 프로그래밍된 펌웨어

TCP/IP 스택: 네트워크 연결을 위해 통합됨

# 일반적인 응용 분야
IoT 센서: 모션 감지 등 다양한 센싱 응용에 사용

Wi-Fi 기능 추가: Wi-Fi가 없는 마이크로컨트롤러에 Wi-Fi 기능 추가

홈 오토메이션: 스마트 홈 기기 및 가전제품 제어

데이터 로깅: 데이터 수집 및 클라우드 서버로 전송

# 프로그래밍 옵션
AT 명령어: 사전 프로그래밍된 펌웨어를 사용한 간단한 통신

Arduino IDE: ESP8266 코어를 사용하여 Arduino 환경에서 프로그래밍 가능

NodeMCU: 더 고급 응용을 위한 Lua 프로그래밍 언어 지원

MicroPython: Python 기반 개발을 위한 MicroPython 실행 가능

# 제한 사항 및 고려 사항
제한된 GPIO: I/O 작업에 사용 가능한 GPIO 핀이 2개뿐임

전원 요구 사항: 안정적인 3.3V 전원 공급 필요

플래시 모드: 플래싱 중 GPIO0을 접지해야 함

발열: 지속적인 작동 시 열이 발생할 수 있음
